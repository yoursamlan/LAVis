Welcome to Lorenz Attractor Visualizer a.k.a LAVis ver 3.0 .

The main AIM of this web application is to visualize the chaotic property of Lorenz Attractor and the effect of parameters to the solution.

First, select Initialize from the OPTION menu. Note down the values of state variables x, y, z and the parameters σ, β, and ρ as the inital set of values. After noting down, select Visualization from the OPTION menu to simulate the Solutions and the Attractor.

If you have any query or need any assistance regarding this project or the application, please feel free to click info and help button from top right corner of this app.

Thank you.


Please use 1080p [1920x1080] Monitor for best viewing experience.

You can use following keys to control the visualization.

MOUSE & KEYBOARD
( left-click + drag ) Rotate X-Y axes
( middle-click + drag ) Translate X-Y axes
( shift + click + drag ) Rotate / Translate Z axis
( mousewheel ) Zoom in / Zoom out
( spacebar ) Pause / Play
KEYBOARD
a Add solution with random initial value
x Clear all solutions
F5 Reset to Default
h Toggle head (point) visibility
[ Decrease tail length
] Increase tail length
